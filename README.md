# Structural & Lexico-semantic Effect Analysis Research - GraphRAG
This repository contains the datasets, code and experimental results for 'Structural & Lexico-Semantic Effect Analysis' research, focused on evaluating query-based GraphRAG approaches.

## Directory and File Overview

* **analysis/** — Consolidated results, visualizations and statistical insights
* **datasets/** — Source data and intermediate processed formats used across experiments
* **evaluation_results/** — Per-model and per-knowledge-graph outputs, including evaluated responses 
* **prompts/** — Prompt templates used to generate responses from the LLMs
* **scripts/** — Helper python scripts and jupyter notebooks utilized for preprocessing, experimentation, evaluation and analysis
* `LICENSE` — License for the repository
* `README.md` — Readme file for the repository
